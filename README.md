# 糖尿病控制不良評估互動網頁

這個 repository 提供可互動填寫的糖尿病控制不良評估與追蹤工具。

## 內容

- `index.html`：互動網頁入口
- `styles.css`：網頁樣式
- `app.js`：即時計分、追蹤紀錄、本機儲存與 CSV 匯出
- `糖尿病控制不良原因評估表_協作追蹤評值版.xlsx`：Excel 版本

## 功能

- 填寫病人基本資料、初次評估日期與預定後續評估日
- 記錄跨職類協作、後續評估日期、改變情況與醫師評論
- 同一病人可新增多筆追蹤紀錄
- 自動計算評值總分與等第
- 可儲存在本機瀏覽器並匯出 CSV

## 驗證

已完成 JavaScript 語法檢查與 Excel 公式錯誤掃描。
